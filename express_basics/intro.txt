
1.Express is a popular web frameowork which is built on top of NodeJS.
2.Express provides flexibility interface while creating http servers,middleware,routing and 
error handling.
3.Express is widely used in all backend services for web apps
4.It can also be used for complex applications

Features of Express:

1.Faster than NodeJs.
2.Middleware.
3.Routing.
4.Template Engine.
5.Debugging.
